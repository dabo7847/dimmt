Mixins

Mit Mixins kann Code recyclet werden. Wie in Funktionen. Man kann sogar Argumente übergeben. Es ist der Hammer! Wie geil das ist, wird man sehen, wenn man es in css-tables beschreibt! Aber so weit sind wir noch nicht. Sei nicht so ungeduldig! Alles zu seiner Zeit. Wir gucken uns ersteinmal ein Stück Code an. Das kann zum beispiel folgendermaßen aussehen: 

@mixin changeling($farbe){
  color: $farbe;
}

#changeText 
  @include changeling(#aabbcc);

Das wird dann compiliert. Was das ist, solltest du eigentlich wissen, wenn nicht, schlag es halt irgendwo nach, meingott. Sei nicht so unselbstständig! Wenn du es nicht weißt, stell dir einen Zauberer vor. Oder ein Marmeladenbrot. Und wenn das fertig compiliert ist, sieht das Ergebnis so aus:

#changeText {
  color: #aabbcc;
}

Und das kann dann jeder Browser lesen. So einfach ist das!
